Fiz esse projeto para estudar e entender sobre a biblioteca FOLIUM

usada para a plotagem de mapas. Neste trabalho fiz tags de lugares onde visitei. Fiz de uma forma simples para tornar o entendimento do iniciante, assim como eu, mais fácil.
Em minha pesquisa pra execução desse projeto, percebi que há exemplos um pouco mais complexos sobre a biblioteca, tive que abrir muitas abas em meu navegador para fazer um compilado de entendimento e deixar isso de uma forma mais simples.
Ao final coloquei um código onde plota um uma visão mais ampla dos continentes.

______________________________________________________________________________________________________

I made this project to study and understand about the FOLIUM library

used for plotting maps. In this work I did tags from places I visited. I did it in a simple way to make the beginner's understanding, just like me, easier.
In my research for the execution of this project, I realized that there are a few more complex examples about the library, I had to open many tabs in my browser to compile an understanding and leave it in a simpler way.
At the end I put a code where it plots a wider view of the continents.
