Fiz esse projeto para estudar e entender sobre a bibliteca FOLIUM

usada para a plotagem de mapas. Neste trabalho fiz tags de lugares onde visitei. Fiz de uma forma simples para tornar o entendimento do iniciante, assim como eu, mais fácil.
Em minha pesquisa pra execução desse projeto, percebi que há exemplos um pouco mais complexos sobre a biblioteca, tive que abrir muitas abas em meu navegador para fazer um compilado de entendimento e deixar isso de uma forma mais simples.
Ao final coloquei um código onde plota um uma visão mais ampla dos continentes.
